# もじもんマスター GitHub Pages

## 画像の差し替え
`assets/` に以下の名前で画像を置くと反映されます。

- `aniki.png`：兄貴キャラクター
- `screen_success.png`：成功・クエスト完了画面
- `screen_parent_gate.png`：保護者確認画面
- `screen_unlock.png`：マスターパワー解放画面

## 追加すると良い画像
- `screen_adventure.png`：ぼうけん画面
- `screen_quest.png`：クエスト管理画面
- `screen_training.png`：しゅぎょう画面

`index.html` 内の placeholder-shot 部分を img タグに差し替えてください。
