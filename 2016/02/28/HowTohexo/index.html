備忘録として残します。

macでサイトジェネレーター “hexo”の初期設定の仕方

まず OS X 用パッケージマネージャー “Homebrew”をダウンロードします。

Homebrewサイト

サイトに記載されている

/usr/bin/ruby -e “$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)”

をターミナルで入力

インストール後 “ brew doctor “でエラーやワーニングが無い事を確認(エラーなど出た場合　エラーメッセージを確認しメッセージに沿ってググッて解決する)

npm をインストールする。
brew install node

hexo インストール
npm install -g hexo

ブログ作成する
hexo init myblog

移動
cd myblog
npm install

ローカルサーバー起動
hexo server

ローカルで確認
http://localhost:4000/

設定

config.ymlをエディタで開き

title: ブログタイトル
language: ja
timezone: Asia/Tokyo

deploy:
type: git
repo: git@github.com:gitID/gitID.github.io.git
branch: master

よく使うコマンド

記事追加
hexo new TestPage

gitへデプロイ
hexo deploy -g
